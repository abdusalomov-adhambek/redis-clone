# redis-clone

A Redis clone built in Go for learning networking, concurrency, persistence, and the RESP protocol.

## Features

- TCP Server
- RESP Parser
- RESP Encoder
- PING
- SET / GET
- DEL
- EXPIRE
- TTL
- EXISTS
- KEYS
- INCR
- MGET
- MSET
- AUTH
- Persistence
- Background Expiration Cleanup

## Run

go run main.go

## Connect

redis-cli -p 8001
